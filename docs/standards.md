# Стандарты проекта MinIO

## Именование ресурсов (Naming)
Все ресурсы в AWS именуются по шаблону:
`minio-<окружение>-<тип_ресурса>-<назначение>`
*Пример: minio-prod-ec2-server*

## Тегирование (Tagging)
Каждый ресурс созданный через Terraform обязан иметь теги:
* **Project**: minio-aws
* **ManagedBy**: terraform
* **Owner**: (имя студента)
